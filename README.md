# Cronos402 MCP Server

Core MCP (Model Context Protocol) server with x402 payment support for Cronos blockchain.

## Features

- x402 payment-gated tool execution
- USDC.e gasless payments via Cronos facilitator
- Native CRO direct payments
- Compatible with Claude Desktop, Cursor, and other MCP clients

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Testing

```bash
pnpm test
pnpm test:coverage
```
